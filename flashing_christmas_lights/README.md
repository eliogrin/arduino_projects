# Flashing Christmas Lights
5 modes, switching between them with one button.

- Arduino Leonardo (or any other) ~5$
- Bread board ~0.5$
- LED strips ~10$
- TIP31c transistor ~1$
- 12V Power adapter ~3$

total: 20$
